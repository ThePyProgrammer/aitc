# Phase 1 -- UI Design Contract

> Visual and interaction contract for the Foundation + App Shell phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (custom Command Horizon design system) |
| Preset | not applicable |
| Component library | none (custom components, zero border-radius, no-line rule) |
| Icon library | Lucide React (stroke-width: 1.5px globally) |
| Fonts | Space Grotesk (headlines, labels, nav), JetBrains Mono (data, IDs, monospace) |

**Source:** CLAUDE.md explicitly rejects shadcn/ui -- "Command Horizon is too custom; would gut and restyle every component." All components are hand-built to the Command Horizon specification from wireframes/vector_terminal/DESIGN.md.

## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, status badge padding |
| sm | 8px | Compact element spacing, nav item icon-to-label gap |
| md | 16px | Default element spacing, card internal padding |
| lg | 24px | Section padding, sidebar group spacing |
| xl | 32px | Layout gaps between major panels |
| 2xl | 48px | Sidebar top padding (below header) |
| 3xl | 64px | Page-level vertical spacing (unused in Phase 1) |

Exceptions:
- Header height: 56px (h-14, from wireframe)
- Sidebar width collapsed: 80px (w-20, from wireframe)
- Sidebar width expanded: 256px (w-64, from wireframe)
- Nav item height: 44px (touch target minimum for sidebar items)
- Radar grid cell: 40px (background-size from wireframe CSS)

## Typography

| Role | Font | Size | Weight | Line Height | Letter Spacing | Usage |
|------|------|------|--------|-------------|----------------|-------|
| Display | Space Grotesk | 20px (text-xl) | 700 (bold) | 1.2 | -0.025em (tracking-tighter) | App title "AERO_CODE_CMD", view titles |
| Heading | Space Grotesk | 14px (text-sm) | 700 (bold) | 1.4 | 0.1em (tracking-widest) | Section headers, sidebar nav labels, all UPPERCASE |
| Label | Space Grotesk | 10px | 400 (regular) | 1.4 | 0.1em (tracking-widest) | Metadata labels, stat labels, all UPPERCASE |
| Data | JetBrains Mono | 12px (text-xs) | 700 (bold) | 1.4 | -0.025em (tracking-tighter) | Agent IDs, file paths, numeric values |
| Data-sm | JetBrains Mono | 10px | 400 (regular) | 1.4 | -0.025em (tracking-tighter) | Secondary data, coordinates, timestamps |
| Body | JetBrains Mono | 14px (text-sm) | 400 (regular) | 1.5 | normal | Terminal-style body text, command palette input |

**Rules:**
- Space Grotesk is always UPPERCASE with wide tracking for labels and nav
- JetBrains Mono is used for all data display, file paths, and code
- Never use pure white (#FFFFFF) for body text; use on-surface-variant (#adaaaa)
- Reserve on-surface (#ffffff) for critical headers and active nav items only

## Color

### Surface Hierarchy (60% -- Dominant)

| Token | Hex | Usage |
|-------|-----|-------|
| surface-container-lowest | #000000 | Radar canvas background, input field fills, deepest layer |
| surface | #0e0e0e | Primary application background, main content area |
| surface-container-low | #131313 | Header bar, sidebar background, tonal shift panels |
| surface-container | #1a1919 | Primary workspace panels, agent manifest sidebar, card containers |
| surface-container-high | #201f1f | Table headers, elevated widget backgrounds |
| surface-container-highest | #262626 | Detail panels, popovers, highest-elevation surfaces |

### Text Colors (30% -- Secondary)

| Token | Hex | Usage |
|-------|-----|-------|
| on-surface | #ffffff | Critical headers, active nav item text (sparingly) |
| on-surface-variant | #adaaaa | Default body text, metadata, inactive labels, table data |
| outline | #777575 | Ghost button borders at 20% opacity |
| outline-variant | #494847 | Ghost borders at 15% opacity, grid lines at 10% opacity |

### Accent Colors (10% -- Signal)

| Token | Hex | Reserved For |
|-------|-----|-------------|
| primary | #8eff71 | Active nav indicator, healthy agent dots, primary CTA button fill, app title, status "healthy" |
| primary-container | #2ff801 | Gradient endpoint for primary button hover glow |
| secondary | #00cffc | Secondary text links, detail view accents, ghost button text, terminal cursor blink |
| tertiary | #ffd16f | Warning/amber state, caution indicators |
| error | #ff7351 | Conflict agent dots, error status badges, destructive action text |
| error-container | #b92902 | Destructive button background on confirm |

**Accent reserved for:** Active sidebar nav border-left (2px primary), agent blip dots on radar, status indicator pulse rings, primary CTA button, stat counter values, app title text. Never decorative.

### The No-Line Rule

Borders are prohibited for sectioning. Boundaries are defined through:
1. **Tonal shifts** -- surface tier changes between adjacent panels
2. **Negative space** -- wide gutters creating "islands" of data
3. **Edge of light** -- container edge defined by background color ending

**Exception:** Ghost borders (outline-variant at 10-15% opacity) are permitted only when accessibility requires a visible boundary. Active sidebar nav items use a 2px primary left border as a signal, not a section divider.

### Glassmorphism (Floating Elements)

For command palette and tooltips:
- Background: surface-variant (#262626) at 60% opacity
- Backdrop filter: blur(20px)
- Border: outline-variant at 10% opacity (ghost border)

## Interaction Contracts

### Sidebar Navigation
- Click nav item: immediate view switch, no page reload (React Router or equivalent client-side routing)
- Active state: primary text + primary left border (2px) + surface-container background
- Hover state: surface-container-high background + primary text, 150ms transition

### Command Palette
- Open: Ctrl+K (Windows) / Cmd+K (macOS)
- Close: Escape key, or click outside
- Behavior: Type-ahead filter of view names (Radar, Tower, Comms, Conflicts)
- Keyboard: Arrow keys navigate results, Enter selects, results highlighted with primary/5 bg
- Appearance: Glassmorphism overlay, centered, 480px max-width, z-index above all content

### System Tray
- Icon: Reflects overall system status (green = healthy, amber = warning, red = conflict)
- Click: Brings app window to foreground
- Phase 1 default state: green (no agents, no conflicts)

### Window
- Default size: 1280x800 minimum
- Resizable: yes
- Title bar: native Tauri window decorations (platform default)
