Inbound/outbound event wiring (Tauri emit names — executor must use these strings verbatim):
- `agent-event-appended` — payload: `AgentEvent`. Emitted on every new agent_events row insert.
- `agent-turn-started` — payload: `{ agentId: String, sessionId: Option<String> }`. Emitted on first stream_event / first assistant chunk.
- `agent-turn-complete` — payload: `{ agentId: String, sessionId: Option<String>, terminalReason: String, isError: bool }`. Emitted on `{type:"result"}`.
- `agent-session-started` — payload: `SessionStartedPayload`. Emitted on `{type:"system", subtype:"init"}` capture.
- `agent-session-ended` — payload: `SessionEndedPayload`. Emitted by supervisor on `child.wait()` return.
- `agent-delivery-updated` — payload: `DeliveryUpdate`. Emitted on queued → delivered, and on `{type:"result"}` → consumed.
- `agent-thread-cleared` — payload: `String (agent_id)`. Emitted from `clear_agent_thread`.
- `agent-events-marked-read` — payload: `String (agent_id)`. Emitted from `mark_agent_events_read`.

Outbound JSONL frame shape (verified empirically in RESEARCH.md Pattern 3):
```json
{"type":"user","message":{"role":"user","content":[{"type":"text","text":"<user content>"}]}}
```
One line per frame, newline-terminated, UTF-8 only. Use `serde_json::to_string(&envelope)` followed by `.push('\n')` — never `format!` (T-10-06 mitigation).

Max message size constant: `pub const MAX_CHAT_MESSAGE_BYTES: usize = 256 * 1024;` (RESEARCH security mitigation; enforced at the Tauri command entry point).

Max per-line cap for parser: `pub const MAX_STREAM_JSON_LINE_BYTES: usize = 1_048_576;` (1 MiB; lines longer than this are logged and skipped).

Tauri AppHandle usage — use `app_handle.emit("<event_name>", &payload)` on an `&tauri::AppHandle<R>` where `R: tauri::Runtime`. Keep generics so tests can run under `tauri::test::MockRuntime`.

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Claude subprocess stdout → AITC parser | Untrusted bytes; may be malformed JSON, truncated mid-line, or exceed size caps |
| Tauri command boundary (`send_chat_message_to_agent`) | Frontend-supplied `content: String`; size capped before stdin write |
| stdin write path | Live subprocess may exit concurrently; BrokenPipe must not hang the sender |
| `agent_events` SQL inserts | Parser-driven, high-frequency; sqlx bind params mandatory |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-10-06 | Injection | outbound JSONL frame | mitigate | `serde_json::to_string(&envelope)` for every frame; never `format!` or string concat. Content is placed in `.content[0].text` so JSON escaping handles quotes/newlines/backslashes safely. |
| T-10-07 | DoS | parser on malformed line | mitigate | Per-line parse in a `match serde_json::from_str(&line)`; on `Err`, `tracing::warn!` and continue.
```rust
use crate::chat_runtime::types::OutboundFrame;
use std::collections::HashMap;
use std::sync::Arc;
use tokio::sync::{mpsc, Mutex};

pub struct LiveSession {
    pub agent_id: String,
    pub session_id: Option<String>,
    pub stdin_tx: mpsc::Sender<OutboundFrame>,
    pub archived: bool,
    pub last_read_at: Option<String>,
}

#[derive(Default)]
pub struct LiveSessionRegistry {
    sessions: Mutex<HashMap<String, LiveSession>>,
}

impl LiveSessionRegistry {
    pub fn new() -> Self { Self::default() }
    pub fn new_arc() -> Arc<Self> { Arc::new(Self::default()) }

    pub async fn register(&self, session: LiveSession) {
        let mut g = self.sessions.lock().await;
        g.insert(session.agent_id.clone(), session);
    }

    pub async fn get_stdin_tx(&self, agent_id: &str) -> Option<mpsc::Sender<OutboundFrame>> {
        let g = self.sessions.lock().await;
        g.get(agent_id)
            .filter(|s| !s.archived)
            .map(|s| s.stdin_tx.clone())
    }

    pub async fn mark_archived(&self, agent_id: &str) {
        let mut g = self.sessions.lock().await;
        if let Some(s) = g.get_mut(agent_id) {
            s.archived = true;
        }
    }

    pub async fn is_archived(&self, agent_id: &str) -> bool {
        let g = self.sessions.lock().await;
        match g.get(agent_id) {
            Some(s) => s.archived,
            None => true,        // no session registered = effectively archived
        }
    }

    pub async fn bind_session_id(&self, agent_id: &str, session_id: String) {
        let mut g = self.sessions.lock().await;
        if let Some(s) = g.get_mut(agent_id) {
            s.session_id = Some(session_id);
        }
    }

    pub async fn session_id_for(&self, agent_id: &str) -> Option<String> {
        let g = self.sessions.lock().await;
        g.get(agent_id).and_then(|s| s.session_id.clone())
    }

    pub async fn mark_read(&self, agent_id: &str) {
        let ts = chrono::Utc::now().to_rfc3339();
        let mut g = self.sessions.lock().await;
        if let Some(s) = g.get_mut(agent_id) {
            s.last_read_at = Some(ts);
        }
    }

    pub async fn last_read_for(&self, agent_id: &str) -> Option<String> {
        let g = self.sessions.lock().await;
        g.get(agent_id).and_then(|s| s.last_read_at.clone())
    }

    pub async fn remove(&self, agent_id: &str) {
        let mut g = self.sessions.lock().await;
        g.remove(agent_id);
    }
}
```

```rust
pub async fn insert_agent_event(
    pool: &SqlitePool,
    agent_id: &str,
    session_id: Option<&str>,
    event_type: &str,
    payload_json: &serde_json::Value,
    approval_request_id: Option<i64>,
    sequence_number: Option<i64>,
    delivery_status: Option<&str>,
) -> Result<AgentEvent, String> {
    // Auto-compute sequence_number if None and session_id Some.
    let seq = match (sequence_number, session_id) {
        (Some(n), _) => Some(n),
        (None, Some(sid)) => {
            let row: Option<(Option<i64>,)> = sqlx::query_as(
                "SELECT MAX(sequence_number) FROM agent_events WHERE session_id = ?"
            )
            .bind(sid)
            .fetch_optional(pool)
            .await
            .map_err(|e| format!("seq lookup: {e}"))?;
            Some(row.and_then(|t| t.0).unwrap_or(0) + 1)
        }
        (None, None) => None,
    };

    let payload_str = serde_json::to_string(payload_json)
        .map_err(|e| format!("payload serialize: {e}"))?;

    let row = sqlx::query(
        "INSERT INTO agent_events \
         (agent_id, session_id, event_type, payload_json, approval_request_id, sequence_number, delivery_status) \
         VALUES (?, ?, ?, ?, ?, ?, ?) \
         RETURNING id, agent_id, session_id, event_type, payload_json, approval_request_id, sequence_number, delivery_status, created_at"
    )
    .bind(agent_id)
    .bind(session_id)
    .bind(event_type)
    .bind(&payload_str)
    .bind(approval_request_id)
    .bind(seq)
    .bind(delivery_status)
    .fetch_one(pool)
    .await
    .map_err(|e| format!("insert agent_event failed: {e}"))?;

    map_agent_event_row(&row)
}

pub async fn list_events_for_agent(
    pool: &SqlitePool,
    agent_id: &str,
    before_id: Option<i64>,
    limit: i64,
) -> Result<Vec<AgentEvent>, String> {
    let effective_limit = limit.clamp(1, 200);
    let rows = match before_id {
        Some(bid) => sqlx::query(
            "SELECT id, agent_id, session_id, event_type, payload_json, approval_request_id, \
                    sequence_number, delivery_status, created_at \
             FROM agent_events WHERE agent_id = ? AND id < ? \
             ORDER BY id DESC LIMIT ?"
        )
        .bind(agent_id)
        .bind(bid)
        .bind(effective_limit)
        .fetch_all(pool)
        .await,
        None => sqlx::query(
            "SELECT id, agent_id, session_id, event_type, payload_json, approval_request_id, \
                    sequence_number, delivery_status, created_at \
             FROM agent_events WHERE agent_id = ? \
             ORDER BY id DESC LIMIT ?"
        )
        .bind(agent_id)
        .bind(effective_limit)
        .fetch_all(pool)
        .await,
    }
    .map_err(|e| format!("list_events_for_agent: {e}"))?;

    let mut out = Vec::with_capacity(rows.len());
    for row in rows.iter() {
        out.push(map_agent_event_row(row)?);
    }
    Ok(out)
}
```

```rust
pub fn spawn_stream_json_reader(
    stdout: ChildStdout,
    agent_id: String,
    sink: mpsc::Sender<StreamEvent>,
) -> tokio::task::JoinHandle<()> {
    tokio::spawn(async move {
        let reader = tokio::io::BufReader::new(stdout);
        let mut lines = reader.lines();
        let mut accumulated_text = String::new();
        let mut last_chunk_at = tokio::time::Instant::now();
        loop {
            tokio::select! {
                line_result = lines.next_line() => {
                    match line_result {
                        Ok(Some(line)) => {
                            if line.len() > crate::chat_runtime::types::MAX_STREAM_JSON_LINE_BYTES {
                                tracing::warn!(agent_id = %agent_id, len = line.len(),
                                               "stream-json line exceeds MAX_STREAM_JSON_LINE_BYTES, skipping");
                                continue;
                            }
                            dispatch_line(&line, &agent_id, &sink, &mut accumulated_text).await;
                            last_chunk_at = tokio::time::Instant::now();
                        }
                        Ok(None) => {
                            // EOF — subprocess closed stdout.
                            let _ = sink.send(StreamEvent::StdoutClosed).await;
                            break;
                        }
                        Err(e) => {
                            tracing::warn!(agent_id = %agent_id, err = %e, "stdout read err");
                            break;
                        }
                    }
                }
                _ = tokio::time::sleep_until(last_chunk_at + std::time::Duration::from_millis(250)) => {
                    // D-17: 250ms idle — flush any buffered assistant text.
                    if !accumulated_text.is_empty() {
                        let _ = sink.send(StreamEvent::AssistantText {
                            content: std::mem::take(&mut accumulated_text),
                            model: None,
                        }).await;
                    }
                    // Reset the timer so we don't spin; wait for the next chunk or infinity.
                    last_chunk_at = tokio::time::Instant::now() + std::time::Duration::from_secs(3600);
                }
            }
        }
    })
}
```

```rust
pub fn spawn_outbound_writer(
    mut stdin: ChildStdin,
    mut rx: mpsc::Receiver<OutboundFrame>,
    delivery: mpsc::Sender<DeliveryUpdate>,
    agent_id: String,
) -> tokio::task::JoinHandle<()> {
    tokio::spawn(async move {
        while let Some(frame) = rx.recv().await {
            let envelope = serde_json::json!({
                "type": "user",
                "message": {
                    "role": "user",
                    "content": [{"type": "text", "text": frame.content}],
                }
            });
            let mut line = match serde_json::to_string(&envelope) {
                Ok(s) => s,
                Err(e) => {
                    tracing::error!(agent_id = %agent_id, err = %e, "envelope serialize failed");
                    let _ = delivery.send(DeliveryUpdate {
                        event_id: frame.event_id, status: "unsupported".into(),
                    }).await;
                    continue;
                }
            };
            line.push('\n');
            match stdin.write_all(line.as_bytes()).await {
                Ok(()) => {
                    if let Err(e) = stdin.flush().await {
                        tracing::warn!(agent_id = %agent_id, err = %e, "stdin flush failed");
                        let _ = delivery.send(DeliveryUpdate {
                            event_id: frame.event_id, status: "unsupported".into(),
                        }).await;
                        break;
                    }
                    let _ = delivery.send(DeliveryUpdate {
                        event_id: frame.event_id, status: "delivered".into(),
                    }).await;
                }
                Err(e) => {
                    tracing::warn!(agent_id = %agent_id, err = %e, "stdin write failed (likely subprocess exit)");
                    let _ = delivery.send(DeliveryUpdate {
                        event_id: frame.event_id, status: "unsupported".into(),
                    }).await;
                    break;
                }
            }
        }
    })
}
```

```rust
pub async fn auto_resume_send<R: tauri::Runtime>(
    agent_id: &str,
    content: &str,
    session_id: Option<&str>,
    cwd: &std::path::Path,
    pool: &sqlx::SqlitePool,
    app_handle: &tauri::AppHandle<R>,
) -> Result<crate::chat_runtime::types::AgentEvent, String> {
    // T-10-09: size cap.
    if content.len() > crate::chat_runtime::types::MAX_CHAT_MESSAGE_BYTES {
        return Err("message exceeds 256 KiB limit".into());
    }
    // T-10-11: validate session_id shape.
    let sid = match session_id {
        Some(s) if is_valid_uuidv4(s) => s.to_string(),
        Some(_) => return Err("invalid session_id shape".into()),
        None => return Err("no session_id available for auto-resume".into()),
    };

    // Insert the user_text row first (D-10 optimistic append).
    let payload = serde_json::json!({ "content": content });
    let user_event = crate::db::events::insert_agent_event(
        pool, agent_id, Some(&sid), "user_text", &payload, None, None, Some("queued")
    ).await?;
    let _ = app_handle.emit("agent-event-appended", &user_event);

    // Insert SESSION_RESUMED boundary.
    let boundary_payload = serde_json::json!({ "kind": "resumed", "session_id": sid });
    let _ = crate::db::events::insert_agent_event(
        pool, agent_id, Some(&sid), "session_boundary", &boundary_payload, None, None, None
    ).await;

    // Spawn `claude --resume <sid> --print <content> --output-format stream-json --verbose`
    let output = tokio::process::Command::new("claude")
        .args(["--resume", &sid, "--print", content, "--output-format", "stream-json", "--verbose"])
        .current_dir(cwd)
        .output()
        .await
        .map_err(|e| format!("auto_resume claude spawn: {e}"))?;

    if !output.status.success() {
        let _ = crate::db::events::update_event_delivery_status(pool, user_event.id, "unsupported").await;
        return Err(format!("claude --resume exit {:?}", output.status.code()));
    }

    // Parse result from last `{type:"result"}` envelope in stdout.
    let stdout_str = String::from_utf8_lossy(&output.stdout);
    let result_line = stdout_str.lines()
        .filter_map(|l| serde_json::from_str::<serde_json::Value>(l).ok())
        .find(|v| v.get("type").and_then(|t| t.as_str()) == Some("result"));
    if let Some(v) = result_line {
        if let Some(text) = v.get("result").and_then(|r| r.as_str()) {
            let assistant_payload = serde_json::json!({ "content": text });
            let _ = crate::db::events::insert_agent_event(
                pool, agent_id, Some(&sid), "assistant_text", &assistant_payload, None, None, None
            ).await;
        }
    }

    let _ = crate::db::events::update_event_delivery_status(pool, user_event.id, "consumed").await;
    Ok(user_event)
}

fn is_valid_uuidv4(s: &str) -> bool {
    // Accept classic hyphenated UUIDv4 shape; don't require v4-specific bit pattern
    // (auto-resume accepts whatever session_id Claude gave us, as long as it looks UUID-shaped).
    let bytes = s.as_bytes();
    if bytes.len() != 36 { return false; }
    for (i, b) in bytes.iter().enumerate() {
        match i {
            8 | 13 | 18 | 23 => if *b != b'-' { return false; },
            _ => if !b.is_ascii_hexdigit() { return false; },
        }
    }
    true
}
```

```rust
use super::session_registry::LiveSessionRegistry;
use super::types::{AgentEvent, ChatChannel, OutboundFrame, MAX_CHAT_MESSAGE_BYTES};
use crate::agents::registry::AgentRegistry;
use sqlx::{Pool, Row, Sqlite};
use std::sync::Arc;
use tauri::Emitter;

fn adapter_chat_duplex(adapter_type: &str) -> bool {
    matches!(adapter_type, "claude-code")
}

#[tauri::command]
#[specta::specta]
pub async fn send_chat_message_to_agent(
    agent_id: String,
    content: String,
    pool: tauri::State<'_, Pool<Sqlite>>,
    sessions: tauri::State<'_, Arc<LiveSessionRegistry>>,
    registry: tauri::State<'_, Arc<AgentRegistry>>,
    app_handle: tauri::AppHandle,
) -> Result<AgentEvent, String> {
    // T-10-09 size cap.
    if content.len() > MAX_CHAT_MESSAGE_BYTES {
        return Err("message exceeds 256 KiB limit".into());
    }
    let agent = registry.get_agent(&agent_id).await
        .ok_or_else(|| format!("unknown agent: {agent_id}"))?;
    let duplex = adapter_chat_duplex(&agent.adapter_type);
    if !duplex {
        // Read-only adapter path: insert unsupported row, no stdin write.
        let payload = serde_json::json!({ "content": content });
        let row = crate::db::events::insert_agent_event(
            pool.inner(), &agent_id, None, "user_text",
            &payload, None, None, Some("unsupported")
        ).await?;
        let _ = app_handle.emit("agent-event-appended", &row);
        return Ok(row);
    }

    let session_id = sessions.session_id_for(&agent_id).await;
    let stdin_tx = sessions.get_stdin_tx(&agent_id).await;
    let payload = serde_json::json!({ "content": content });

    match stdin_tx {
        Some(tx) => {
            let row = crate::db::events::insert_agent_event(
                pool.inner(), &agent_id, session_id.as_deref(), "user_text",
                &payload, None, None, Some("queued")
            ).await?;
            let _ = app_handle.emit("agent-event-appended", &row);
            let frame = OutboundFrame { event_id: row.id, content };
            if let Err(e) = tx.send(frame).await {
                tracing::warn!(agent_id = %agent_id, err = %e, "outbound enqueue failed");
                let _ = crate::db::events::update_event_delivery_status(
                    pool.inner(), row.id, "unsupported"
                ).await;
            }
            Ok(row)
        }
        None => {
            // Archived or never-registered session → auto-resume.
            let cwd = agent.cwd.clone().unwrap_or_else(|| std::env::current_dir().unwrap_or_default());
            crate::chat_runtime::auto_resume::auto_resume_send(
                &agent_id, &content, session_id.as_deref(),
                std::path::Path::new(&cwd), pool.inner(), &app_handle
            ).await
        }
    }
}

#[tauri::command]
#[specta::specta]
pub async fn list_chat_channels(
    pool: tauri::State<'_, Pool<Sqlite>>,
    sessions: tauri::State<'_, Arc<LiveSessionRegistry>>,
    registry: tauri::State<'_, Arc<AgentRegistry>>,
) -> Result<Vec<ChatChannel>, String> {
    let agents = registry.list_agents_detailed().await;
    let mut out = Vec::with_capacity(agents.len());
    for info in agents {
        let adapter_type = info.adapter_type.clone();
        let status = format!("{:?}", info.state).to_lowercase();
        let archived = sessions.is_archived(&info.id).await;
        let chat_duplex = adapter_chat_duplex(&adapter_type);
        let current_session_id = sessions.session_id_for(&info.id).await;
        let last_event: Option<AgentEvent> = {
            let rows = sqlx::query(
                "SELECT id, agent_id, session_id, event_type, payload_json, approval_request_id, \
                        sequence_number, delivery_status, created_at \
                 FROM agent_events WHERE agent_id = ? ORDER BY id DESC LIMIT 1"
            )
            .bind(&info.id)
            .fetch_optional(pool.inner())
            .await
            .map_err(|e| format!("last_event lookup: {e}"))?;
            match rows {
                Some(row) => Some(crate::db::events::map_agent_event_row(&row)?),
                None => None,
            }
        };
        let last_read = sessions.last_read_for(&info.id).await;
        let unread_count: i64 = {
            let row: Option<(i64,)> = sqlx::query_as(
                "SELECT COUNT(*) FROM agent_events WHERE agent_id = ? AND created_at > ?"
            )
            .bind(&info.id)
            .bind(last_read.unwrap_or_else(|| "1970-01-01".into()))
            .fetch_optional(pool.inner())
            .await
            .map_err(|e| format!("unread_count: {e}"))?;
            row.map(|t| t.0).unwrap_or(0)
        };
        out.push(ChatChannel {
            agent_id: info.id,
            adapter_type, status, archived, chat_duplex,
            last_event, unread_count, current_session_id,
        });
    }
    Ok(out)
}
```
