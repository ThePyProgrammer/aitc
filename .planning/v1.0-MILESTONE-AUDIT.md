## Critical Integration Gaps

### 1. Pipeline Activation (FMON-01/02/03/04)

**Impact: HIGH** — The `usePipelineChannel` hook (Phase 2) is never imported or called from any UI component. This means:
- File watcher never starts
- No live file events flow to the frontend
- Radar treemap shows empty state permanently
- Conflict detection pipeline never activates from UI
- Heat map has no data source

**Fix:** Call `usePipelineChannel().register(repoRoot)` from TowerControl or a top-level App component on mount. This is a single-line integration — all backend infrastructure is wired.

### 2. Passive Agent Detection (AGNT-03)

**Impact: MEDIUM** — `ProcessSnapshot.candidates()` never bridges to `AgentRegistry`. Externally-launched agents only appear via self-registration HTTP endpoint (port 9417). Process scanning exists but results never flow to the registry.

**Fix:** Add a periodic bridge task that reads `ProcessSnapshot` candidates and upserts into `AgentRegistry`.

### 3. Session File Tracking (HIST-01)

**Impact: LOW** — `record_session_file` command registered but never called. SessionsTab expandable rows will show empty file lists.

**Fix:** Call `record_session_file` from the pipeline event handler or from the watcher broadcast task.
